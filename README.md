# eu-te-escuto-v2
Eu Te Escuto V2 
Projeto web desenvolvido como evolução da versão inicial "Eu Te Escuto", com foco em melhoria de layout, organização de código e aplicação prática de conceitos de desenvolvimento front-end.





## 🚀 Objetivo

Criar uma aplicação web estática com foco em apresentação de conteúdo e experiência do usuário, aplicando fundamentos de HTML e CSS.

## 🛠️ Tecnologias utilizadas

* HTML5
* CSS3

## 📈 Evolução em relação à versão anterior

* Melhor organização da estrutura HTML
* Separação adequada entre HTML e CSS
* Layout aprimorado
* Ajustes visuais para melhor experiência do usuário

## 🎯 Competências desenvolvidas

* Estruturação de páginas com HTML semântico
* Estilização com CSS
* Organização de arquivos front-end
* Construção de layouts simples
* Versionamento com Git

## 📌 Status do projeto

Em desenvolvimento — futuras versões incluirão interatividade e melhorias na experiência do usuário.
